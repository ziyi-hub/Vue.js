# app-cli-router

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# problem de liens relatives

Retirer lle / devant '/dist/build.js'

```
